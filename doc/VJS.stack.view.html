public
threejsslice(origin, direction)

Create a slice from stack provided to constructor. Call stackModel.prepare() before to make sure the model is ready to be viewed. Makes current view threeJS friendly.

Example 

var stackModel = ...;
stackModel.prepare();

var stackView = new VJS.stack.view(stackModel);

var direction = new THREE.Vector3(0, 0, 1);
var origin = new THREE.Vector3(448, 448, 32);
stackView.threejsslice(origin, direction);

scene.add(stackView);
