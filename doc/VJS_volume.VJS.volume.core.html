Class: VJS.volume.core

VJS.volume.core

core

new VJS.volume.core(data, max, min, transforms, ijk, ras, nbTexture, maxTextureSize)

Core of volume module. Holds all information related to the volume. Provides core volume-related functionnalities. Doesn't deal with the visualization of the volume.
Parameters:
Name Type Description
data Array Raw data that represents the volume
max Number Maximum intensity in the volume. Used for data normalization. (should probably by rgb array...)
min Number Minimum intensity in the volume. Used for data normalization. (should probably by rgb array...)
transforms Object All volume related transforms, such as ijk to ras.
ijk Object All ijk related information, such as dimensions, origin, etc.
ras Object All ras related information, such as dimensions, origin, etc.
nbTexture Number Number of textures to be generated from the raw data.
maxTextureSize Number Size of each texture to be generated from the raw data.
