## Dev corner

Default task (runs tests, documentation, etc. and create a directory ready to be pushed as gh-page)
```
$> gulp
```

For development (runs test tests and reloads page on every change)
```
$> gulp serve
```

Build standalone library to build/
```
$> gulp build
```

Tests
```
$> gulp test
```

Documentation
```
$> gulp doc
```
